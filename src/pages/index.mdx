# System Design & OOD Interview Prep

Welcome to the ultimate resource for cracking system design and object-oriented design interviews. This platform covers everything from fundamental concepts to deep-dive implementations with interactive MCQs.

Use the **sidebar on the left** to navigate between topics. Use the **search bar** at the top to find any topic instantly. Each page includes:

- **Interactive MCQ quizzes** at the end to test your understanding

### OOD Interview Problems (10 Problems)
Practice with full Python implementations, class diagrams, and pattern-based solutions:
- **Library Management** — Observer (notifications), Strategy (fee calculation)
- **Vending Machine** — State pattern (Idle, HasMoney, Dispensing)
- **ATM System** — Chain of Responsibility (bill dispensing), State (transaction flow)
- **Elevator System** — Strategy (SCAN/Nearest scheduling), Observer (floor displays)
- **Online Bookstore** — Factory (payments), Builder (order construction), Observer (notifications)
- **Notification System** — Factory + Decorator (retry, logging, rate limiting)
- **File System** — Composite (file/directory tree), Iterator (DFS/BFS traversal)
- **Ride-Share Billing** — Strategy (pricing), Decorator (discounts, tips, tolls)
- **Chat Room** — Mediator (room mediates users), Observer (presence tracking)
- **Rate Limiter** — Strategy (Token Bucket, Sliding Window, Fixed Window), Factory
